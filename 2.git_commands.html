Running Git in a Project

If you are running git for the first time then in terminal of you system you need to run a command.

Lets Take a Look at Steps and Commands

  1. Installing Git SCM on windows with visual studio as default editor. Then you can run in terminal cmd or powershell
  2. Connecting Accounts from here to github ( to be done 1 time only to configure git in system)
    1. git config --global user.name "dhillongeetdev"
    2. git config --global user.email "dhillongeetdev@gmail.com"
  3. Initializing a folder with git in windows and connecting with the right repository
    1. git init (run in the same folder you want to comit git in)
    2. git remote add origin "https://github.com/dhillongeetdev/geetnew.git" ( Code>Local>Https in github account)
  4. Pull Request ( to pull data from git hub to computer from a branch)
    1. git pull origin main ( main is the branch name)
  5. Push from computer to github in a branch
    1. git status ( to check what is pending)
    2. git add . (. is for all files, i can write file names here 1 after the other)
    3. git status
    4. git commit -m "any message here"
    5. git push
  6. creating a new branch
    1. git checkout -b "day1Branch" (day1Branch can be replaced by the new branch name you want to create)
  7. switching to old branches
    1. git checkout "main" (branch name)
  8. Cloning a project
    1. git clone https://github.com/dhillongeetdev/geetnew.git ( the url from code>local>https and the clone you need to open the folder where you want to clone this github branch to the computer folder, so in the computer folder open cmd and use this command)
